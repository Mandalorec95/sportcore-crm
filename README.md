# SportCore CRM

CRM-система для детских спортивных секций и клубов.

Проект для хакатона: CRM для выбранной ниши с публичным API для внешних интеграций.

## Ниша

Детские спортивные секции, клубы единоборств, футбольные школы, секции гимнастики, плавания, борьбы, бокса, тхэквондо, каратэ и танцев.

## Главная идея

Не просто CRM для записи на тренировки, а цифровой паспорт спортсмена: посещаемость, оплаты, медсправки, соревнования, прогресс и связь с родителями.
